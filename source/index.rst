CUDA 编程指南
==============

**CUDA 与 CUDA 编程指南**

CUDA 是 NVIDIA 开发的并行计算平台和编程模型，通过利用 GPU 的强大能力实现计算性能的显著提升。
它允许开发者加速计算密集型应用程序，广泛应用于深度学习、科学计算和高性能计算 (HPC) 等领域。

本 CUDA 编程指南是关于 CUDA 编程模型以及如何使用 CUDA 平台在 GPU 上编写代码的官方综合资源。
本指南涵盖了从 CUDA 编程模型和 CUDA 平台到语言扩展细节的所有内容，并介绍了如何利用特定的硬件和软件功能。
本指南为新接触 CUDA 的开发者提供了一条学习路径，同时也为使用 CUDA 构建应用程序的开发者提供了重要的参考资料。

**本指南的组织结构**

即使对于主要使用库、框架或 DSL 的开发者，了解 CUDA 编程模型以及 GPU 如何执行代码对于理解抽象层背后发生的事情也很有价值。
本指南首先介绍了 :ref:`编程模型<programming-model>`，该章节不涉及任何特定编程语言，适用于任何想要了解 CUDA 如何工作的人，包括非开发者。

本指南分为五个主要部分：

- 第一部分：简介和编程模型概述

  - 与语言无关的 CUDA 编程模型概述以及 CUDA 平台的简要介绍。

  - 本章节适合任何想要了解 GPU 和在 GPU 上执行代码概念的人阅读，即使他们不是开发者。

- 第二部分：使用 CUDA 编程 GPU

  - 使用 CUDA C++ 编程 GPU 的基础知识。

  - 本章节适合任何想要开始 GPU 编程的人阅读。

  - 本章节旨在教学而非完整，教授 CUDA 编程中最重要和最常见的部分，包括一些常见的性能考虑因素。

- 第三部分：高级 CUDA

  - 介绍 CUDA 的一些更高级的功能，这些功能既能实现细粒度控制，又能提供更多最大化性能的机会，包括在单个应用程序中使用多个 GPU。

  - 本章节最后是 :ref:`CUDA 功能概览<tour-of-features>`，简要介绍了第四部分中涵盖的每个功能的用途和功能，按开发者何时以及为何会发现每个功能有用进行排序。

- 第四部分：CUDA 功能

  - 本章节包含对特定 CUDA 功能的完整介绍，如 CUDA 图、动态并行、与图形 API 的互操作性以及统一内存。

  - 当需要了解特定 CUDA 功能的完整图景时，应参考本章节。在可能的情况下，本章节中涵盖的功能已在早期章节中进行了介绍和动机说明。

- 第五部分：技术附录

  - 技术附录提供了关于 CUDA 的 C++ 高级语言支持、硬件特定规范和其他技术规范的参考文档。

  - 本章节作为技术参考，用于特定描述 CUDA 元素的语法、语义和技术行为。

第一至三部分为 CUDA 新手开发者提供了引导式学习体验，同时也为任何经验水平的 CUDA 开发者提供了有用的见解和更新信息。

第四和第五部分提供了关于特定功能和详细主题的大量信息，旨在为需要在编写 CUDA 应用程序时了解更多细节的开发者提供精心策划、组织良好的参考。
